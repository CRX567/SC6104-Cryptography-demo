次哈希计算。

---

### 2. 传统方法：生日攻击

生日攻击的核心思想是：
1. 随机生成大量输入；
2. 计算它们的哈希；
3. 比较是否存在相同结果。

虽然这种方法时间复杂度低（约 `O(2^(m/2))`），  
但需要存储所有哈希结果（空间复杂度也为 `O(2^(m/2))`），  
因此在哈希长度较大时（例如 40 位、64 位）几乎不可行。

---

### 3. Pollard’s Rho 算法思想

Pollard’s Rho 算法由 John Pollard 在 1975 年提出，最初用于整数分解。  
它利用了一个简单的事实：

> 在一个有限集合上重复应用一个确定性函数，  
> 最终必然进入循环（cycle）。

这种“尾部 + 环”的结构形状类似希腊字母 ρ（Rho），算法因此得名。
